Next-generation SIP Server

Lightweight sip proxy, location server, and registrar that provides a reliable and scalable SIP infrastructure for telephony carriers, communication service providers, and integrators.
Get Started


battery_charging_full

Batteries Included

Routr ships with all the tools necessary to deploy your VoIP network. A command-line is included to control the server remotely. You can also control the server using the RESTful API or the GUI from the comfort of the web browser.

cloud

Cloud Ready

Our team maintains a docker image that contains the server, and all the necessary tools to deploy Routr in few seconds. The image is periodically tested agains the most common cloud providers. Our goal is to make Routr a cloud first-class citizen.

We have provided detailed documentation as well as specific code examples to help new users get started. We are also always open to feedback and can answer any questions a user may have about Routr.
